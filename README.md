# YKImageCropper

## Installation

```ruby
pod 'YKImageCropper', '~> 0.0.1'
```

## Usage

```objective-c
UIImage *image = [UIImage imageNamed:@"Sample.jpg"];
YKImageCropperViewController *vc = [[YKImageCropperViewController alloc] initWithImage:image];
```
